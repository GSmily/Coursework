To Start:
First click on the maze size you want to play "5x5", "10x10", etc.
To repeat the same size click "New Maze"
To change sizes just click a different size button.

Controls:
w: move up
a: move left
s: move down
d: move right
h: Hint toggle
b: Bread toggle
p: Path toggle

Game Scoring:

5 points for each cell in shortest path
-2 points for any other cell
-20 points for using path (p)
-5 points for using hint (h)
